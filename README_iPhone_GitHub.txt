A4写真報告書 iPhone/GitHub Pages版

変更内容:
- Android/Capacitor用の保存処理を削除
- 途中保存、保存読込、PDF保存、印刷、完了保存を削除
- JPEG保存だけ残しました
- iPhoneではJPEG保存ボタンを押し、共有画面から「画像を保存」を選ぶと写真フォルダへ保存できます

GitHub Pagesで使う方法:
1. このZIPを展開
2. 中の index.html / app.js / styles.css をGitHubリポジトリへアップロード
3. GitHub Pagesを有効化
4. iPhoneのSafariでGitHub PagesのURLを開く

注意:
- 写真フォルダへの自動保存はiPhone/Safariの制限でできません
- 共有画面で「画像を保存」を選ぶ操作が必要です
- GitHub PagesはHTTPSなので共有保存に対応しやすいです
